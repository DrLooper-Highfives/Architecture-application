Calcul mental

Comment jouer ?

  • Au début du jeu, par défaut, vous avez une opération d'addition ou soustraction à 2 chiffres
  • Vous pouvez choisir de changer de type d'opération
  • Cliquez sur le bouton "Nouvelle partie"
  • Dans le premier cadre gris apparait une opération
  • Saisissez le résultat dans le 2ème écran gris et validez avec la touche "entrée"
  • Si le résultat est bon, le score s'ajoute de la position dans la série de bonnes réponses que vous avez obtenues (si vous venez de saisir le 3ème résultat d'affilé bon, le score s'ajoutera de 3 pts)
  • Si le résultat est mauvais, le score se déduira d'1 point sauf s'il est nul. De plus, le prochain ajout dans le score ne sera que d'1 pt, sauf si vous avez 0 pt.
  • Vous avez 2 mn pour faire le score maximum
  • À la fin du temps, l'écran change et vous propose une nouvelle partie. Le score final reste et la note sur 20 est donnée
Ce jeu s'adresse à tout le monde. Il y en a pour tous les niveaux. École primaire (table addition et soustration), collégiens et lycéens (pour les autres items), ou adultes.
A chaque fin de partie, une note sur 20 est donnée vous donnant votre niveau. La note de 20 peut être obtenue si vous êtes suffisamment entraîné.
